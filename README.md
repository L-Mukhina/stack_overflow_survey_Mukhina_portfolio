# Working with Large Datasets (Stack Overflow Data Analysis)

Description:  
This project analyzed large datasets from Stack Overflow to explore trends in the IT industry.  

Tools:  
Python, Pandas  

My Tasks:  
- Applied descriptive statistical methods to summarize data  
- Calculated key metrics such as age categories, education, and industry insights  
- Investigated Python popularity and trends among high-paid IT specialists  
- Created aggregated tables to present results clearly  

Result:  
- Obtained insights on work experience, remote work trends, learning paths, and programming language popularity  
- Analysis saved in Jupyter Notebook (.ipynb) format
